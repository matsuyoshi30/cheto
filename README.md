# cheto

A simple pomodoro timer for macOS. Lives in your menu bar with an optional floating overlay window.

## Features

- Menu bar integration with real-time countdown display
- Floating always-on-top timer window (draggable, position remembered)
- 25 min work / 5 min break cycle
- Notification + sound on phase completion
- Color change on floating window when phases transition

## Build & Run

```
make run
```

Requires macOS 14+ and Swift 5.10+.
